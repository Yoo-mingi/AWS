서버 내부에 연결하는 것.
사용자의 운영체제에 따라 다르지만 대부분 SSH는 CLI로 MAC, Linux에서 가능하고 window10이상부터 가능하다.
window 10이하라면 putty를 사용해야한다.
마지막으로 웹 브라우저에서 연결이 가능한데, amanzon 홈페이지 접근이 가능하다면 다 된다.

pem파일을 사용하여 ssh접속을 진행할것이다.
원하는 파일에 집어넣고 공용IP를 잘 적어놓은 이후에 보안그룹에 22번 포트가 열려있는지 확인한다.

# ssh ec2-user@EC2인스턴스의IP주소

안된다면
# ssh -i pem파일 ec2-user@EC2인스턴스의IP주소

EC2 인스턴트 커넥터.
EC2 선택하고 우측 상단에 connect선택. User name을 입력하고 접속한다.
ssh key가 필요없는데 임시키를 발급하고 접속하기 때문. 관리 필요없어서 편하다.
